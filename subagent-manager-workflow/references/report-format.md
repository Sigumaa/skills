# 報告フォーマット規約

## 中間進捗報告（固定項目）
- Proposed Plan 正本化状況（作成/更新、検証結果）
- 対象フェーズ / 対象項目
- 並列実行中タスク（task_id, 担当, 状態）
- 待機中タスク（待ち合わせ条件つき）
- 失敗/ブロッカー（失敗タイプ, 対応状況）
- ユーザー相談の要否（gate_id）
- 次の更新予定

## 最終報告（固定項目）
- 完了スコープ
- Proposed Plan 正本情報（対象ファイル、更新要約、検証結果）
- 変更ファイル一覧（global/repo別）
- 委譲実績（実装/レビュー/テスト/コミットの担当）
- 検証結果（実行コマンドと結果）
- 失敗復旧履歴（発生時のみ）
- ユーザー相談履歴（発生時のみ）
- コミット情報（対象repoのみ）
- 残課題 / 次アクション

## 記載テンプレート
```text
[中間進捗]
- Proposed Plan:
- 対象フェーズ/項目:
- 並列実行中:
- 待機中:
- 失敗/ブロッカー:
- 相談要否:
- 次更新:
```

